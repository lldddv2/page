Velocidad: 0.000
Tiempo: 0.000
RA: 0.000
DEC: 0.000

Controles:
up/down: RA
left/right: DEC
ctrl + up/down: Velocidad
ctrl + left/right: Tiempo
space: Pausar/Reanudar Tiempo

--- ver/oculta objetos ---
Cada objeto tiene una posición
inicial natural.
Alt + C: el dado (Terrell).
Alt + E: la Tierra (Deformación).
Alt + J: Jupiter (Deformación).

---- Recomendaciones -----
Para el efecto de rotación de
Terrell, se recomienda estar
apuntando a la posición inicial.
El efecto de aberración está incluido.
